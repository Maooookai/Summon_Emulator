# 抽卡模拟器
_本插件基于[JCQ-CoolQ](https://gitee.com/meowy/JCQ-CoolQ)制作_

当前版本1.0.6

目前支持阴阳师和明日方舟的抽卡模拟

阴阳师发送“抽卡”或“十连”即可使用

明日方舟的指令为“寻访”或“十连”
